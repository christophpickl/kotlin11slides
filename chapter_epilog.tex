\begin{frame}[fragile] \frametitle{Logging the Kotlin way}

First declare a Gradle Dependency:
\begin{lstlisting}
compile
 "io.github.microutils:kotlin-logging:1.4.4"
\end{lstlisting}
\pause

Write your own shortcut function:
\begin{lstlisting}
fun LOG(func: () -> Unit) =
  KotlinLogging.logger(func)
// define a code template in your IDE
\end{lstlisting}
\pause

Simple usage:
\begin{lstlisting}
class Foo {
  private val log = LOG {}
  init {
    log.debug { "lazy evaluated $this" }
  }
}
\end{lstlisting}
\pause

\end{frame}
